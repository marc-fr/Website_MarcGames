About the game

The player pilots the lunar module and the goal is to reach the arrival platform before running out of fuel. Take up the challenge to pilot safely to the destination.

Play the demo version on web: ichtio page

Features

  • Build your skills to pilot in a resistless context
  • Admire the accurate modeling of the Moon and the lunar module from Apollo 11
  • Feel the mass of the lunar module and the low gravity on the Moon surface
  • Create your own levels thanks to the level editor

Development log

The game is currently released on the version Beta 7. The next release is planned to be the first finalized version. No release date has been fixed.

Release beta 7 (Jan 2023)

From this version, I fixed feedback reported by some testers.

Release beta 5 and 6 (August 2022)

For those versions, I focused to implement all side things that improve the user experience, such as adding check-points and achievements, adding the support of joysticks, ... A level editor was added also that allows the player to edit custom levels.

Release beta 4 (May 2020)

Lot of improvements have been made on the models and the rendering. Lot of efforts have been put for the moon modeling.

Release beta 3 (Nov 2018) and before

Those versions follow drafts of the game concept, with basic rendering and placeholder models. It was my first game using C++ / OgenGL, so this game was a good case for learning rendering. The physics part is good enough, to start to feel the game-play.

Prototype (~2003)

The initial idea was born in my math class and was implemented on the programmable calculator TI-82. The program is lost unfortunately. The game was a 1D, in which you control the engine thrust while the vessel is in space and attracted by the moon's gravity.
