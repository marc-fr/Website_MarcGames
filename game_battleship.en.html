About the game

The player controls battleships, inspired from the World-War 2. The game is currently an early prototype, with a playable alpha demo on web. There is still lot of work, notably game-play and special-effects.

  • Play epic battles, with a challenging AI
  • Fine control of aiming
  • Dynamic weather conditions

Release history

Release Alpha 1 (2025 January):

  • Chnaged the way to aim (less arcade feeling)
  • Created collision-meshes and shells hit
  • Added shadows and SSOA
  • Added smoke and water splashes
  • Improved cloud rendering with per-example noise from specialized textures
  • Added game options (but not visible on the web demo)
  • Fixed simulation instabilities with high frame-rate
  • Implemented a first and quick health system
  • Improved night looking (post-effects)

Protoptye (2024 July):

  • Improved procedural sky rendering
  • Improved overall rendering (ocean shaders, clouds, added bloom effect and light reflections)
  • Implemented a first version of the weather system
  • Started to model ships
  • Added first version of the stars rendering, with a reallistic sky-dome motion and sun path during the day
  • Implemented the first UI elements
  • Implemented ballistics computation

Protoptye (2023 April):

  • First version of the procedural sky rendering
  • Build the game's core with placeholders
  • Design the camera control

Protoptye (2010's):

Made with python, the game concept is inspired by the RTS game gendra. The navy played an important role during the early Word-War 2, before that the aviation becomes pre-dominant over the seas.
